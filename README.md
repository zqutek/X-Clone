# X Clone

Навчальний клон X / Twitter на React Native, Expo Router, Clerk та Convex.

## Технології

- React Native
- Expo
- Expo Router
- Clerk
- Convex
- TypeScript

## Реалізовано

- базова структура Expo Router;
- темна тема;
- tab navigation;
- підключення Clerk;
- підключення Convex;
- Convex schema для основних сутностей;
- базові placeholder-екрани;
- підготовка Clerk webhook для створення користувача в Convex.

## Встановлення

```bash
npm install
npx convex dev
npx expo start
```
